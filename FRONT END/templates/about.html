About DFU Classifier

Dense-ShuffleGCANet: Smart Diabetic Foot Ulcer Detection

Dense-ShuffleGCANet is a deep learning-based system designed for the accurate classification of Diabetic Foot Ulcers (DFU), combining DenseNet-169 and ShuffleNet with a novel attention mechanism called CCDGS (Channel-wise Cross-Domain Global Spatial) attention. This hybrid model enhances feature learning for precise ulcer type identification. Trained on the Kaggle DFU dataset, it achieves high accuracy with fast inference. A Flask-based web application allows users to upload foot images and get predictions in real-time through an HTML, CSS, and JavaScript-powered interface. The system aims to support early DFU detection and reduce complications through smart medical imaging.
